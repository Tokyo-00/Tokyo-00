## 👤 About Me

I'm a **Python developer** from Bangladesh 🇧🇩, operating under the brand **Mr-SxR** which stands for **Speciality & Reliability**. My core focus is building reliable automation tools and desktop software using Python. I also work with reverse engineering — especially Android APKs — and develop full-stack web applications when needed.

I believe in writing software that doesn't just work, but works consistently and dependably. Every project I release under Mr-SxR carries that standard.

- 🔭 Currently building automation tools & desktop apps with Python
- 🔬 Exploring deeper into Android reverse engineering & SSL unpinning
- 💬 Ask me about Python automation, APK analysis, or web development
- 📍 Based in Bangladesh
